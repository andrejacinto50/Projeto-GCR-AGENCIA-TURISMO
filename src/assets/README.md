# Assets

Coloque aqui as fotos e vídeos reais dos passeios (hero, galeria, cards).

Sugestão de organização:
```
assets/
  hero/
  passeios/
    lancha/
    escuna/
    trilhas/
    buggy/
    parapente/
    pesca/
    city-tour/
    banana-boat/
  galeria/
```

Até que as imagens reais sejam adicionadas, o site usa gradientes e formas em
SVG/CSS no lugar de fotos (Hero, Gallery), para manter a identidade visual
sem depender de arquivos externos.
